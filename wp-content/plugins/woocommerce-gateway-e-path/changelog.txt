*** WooCommerce - e-Path Payment Gateway Changelog ***

2018.05.24 - version 1.7.0
 * WooCommerce 3.4 compatibility. This release is also compatible with WooCommerce 3.3.x and 3.2.x.
 * Remove support for WooCommerce versions prior to version 3.0.0.
 * GDPR/Privacy policy support.

2016.01.28 - version 1.6.2
 * Readme updates, including WooCommerce 2.5 compatibility.

2016.01.21 - version 1.6.1
 * Compatibility with older WooCommerce versions (2.3 and 2.2).

2015.12.18 - version 1.6
 * Change: Use order currency rather than store-wide currency for better support with multi-currency WooCommerce extensions.
 * Change: Use WC_Order::get_formatted_billing_full_name() for better compatibility with other WooCommerce extensions.
 * Add WooCommerce 2.4 compatibility. This release is also compatible with WooCommerce 2.3.x.
 * Remove support for WooCommerce v2.1.x or older. This extension now requires WooCommerce v2.2.0 or newer.
 * Documentation/URL updates.

2014.01.22 - version 1.5.0
 * WooCommerce 2.1 compatibility. This release is also compatible with WooCommerce 2.0.x.

2013.07.22 - version 1.4.0
 * Multi-currency support: e-Path now works with any currency that WooCommerce supports!
 * Improved handling of cases where e-Path doesn't reconstruct the return URL correctly.
 * Updated documentation links.

2013.01.07 - version 1.3.0
 * WooCommerce 2.0 compatibility. This release is also compatible with WooCommerce 1.6.x.

2012.12.05 - version 1.2.2
 * Updater

2012.07.10 - version 1.2.1
 * Enhancement to automatically correct thank you page URLs for cases where a customer's e-Path payment gateway isn't configured/customized to work with WooCommerce.

2012.07.09 - version 1.2
 * Important: When upgrading to v1.2, you must contact e-Path support and inform them that your e-Path payment gateway is being used with WooCommerce. e-Path will then modify your gateway to ensure that it works with WooCommerce.
 * Return URL and description parameter fixes (requested by e-Path).
 * Fix fatal error on order received page.

2012.06.28 - version 1.1
 * Stock reduction when order is onhold.
 
2012.05.10 - version 1.0
 * Initial Release.
